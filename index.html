LonelyLessAustralia Decision Aid Tool

This tool helps you evaluate program configurations for reducing loneliness among older adults in Australia.

Loneliness affects over five million Australians, particularly older adults (65 and above), causing significant health and economic challenges. This tool uses discrete choice experiment estimates to predict programme uptake probabilities, enabling personalised services based on the factors older adults value most.

  • Instructions – Step-by-step guidance on using the tool.
  • Attributes – Detailed explanation of programme features (hover over info icons for extra details).
  • Configuration – Configure your scenario parameters using interactive cards and inputs.
  • WTP Results – View charts showing willingness-to-pay estimates for each attribute.
  • Predicted Uptake – See the predicted uptake percentage with tailored recommendations.
  • Costs & Benefits – Analyse cost components and compare QALY-based and WTP-based benefits.
  • Saved Scenarios – Compare and export your saved configurations to PDF.
  • Video Guide – Watch/read a guided tutorial on how to use the tool and interpret results.

How to Use This Tool

  1. Review the Attributes tab to understand each programme feature. Hover over the info icons for details and reference levels.
  2. Go to the Configuration tab and set up your scenario. Select levels for each feature using the cards. (All features except method are required; if no method is selected, in-person is assumed.)
  3. Provide a scenario name and any notes for your own reference (optional).
  4. Click Apply Configuration to calculate results. This will update all result tabs (WTP, Uptake, Costs/Benefits) with the new scenario.
  5. For a quick summary, click View Results Summary. A pop-up will show key outcomes (uptake, net benefit) and recommendations.
  6. Examine the WTP Results tab for monetary values assigned to each attribute (how much participants value each feature).
  7. Check the Predicted Uptake tab for the uptake percentage. Read the insights on how to possibly improve it.
  8. Review the Costs & Benefits tab for a detailed breakdown. Use the QALY gain selector to see different benefit scenarios and compare QALY-based vs WTP-based net benefits.
  9. Save scenarios of interest using Save Scenario. Then use the Saved Scenarios tab to compare them side by side or export to PDF for reporting.
  10. Visit the Video Guide tab for a narrated walkthrough of the tool if you need further guidance or prefer a visual tutorial.

Willingness-to-Pay (WTP) Results

The willingness-to-pay for each programme attribute is calculated by dividing the attribute’s coefficient by the (negative) cost coefficient. This shows the monetary value respondents place on each attribute, holding other factors constant.

For example, among the programme types, community engagement is the most valued – older adults are willing to pay about A$14.5 more per session for community-based programmes (vs peer support). Conversely, they would require compensation to accept a fully virtual delivery (negative WTP of about A$11.7, vs in-person).

Modelling Assumptions

The analysis combines fixed costs (incurred regardless of uptake) and variable costs (scaling with participation). Benefits are calculated in two ways: via QALY gains (valued at A$50k each) and via participants’ willingness-to-pay. Toggle the breakdowns below for more details.
