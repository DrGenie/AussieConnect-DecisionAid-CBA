L LonelyLessAustralia Decision Aid Tool

This tool helps you evaluate programme configurations for reducing loneliness among older adults in Australia.

Loneliness affects over five million Australians, particularly older adults (65 and above), causing significant health and economic challenges. This tool uses discrete choice experiment estimates to predict programme uptake probabilities, enabling personalised services based on the factors older adults value most.

  • Instructions – Step-by-step guidance.
  • Attributes – Detailed explanation of programme features (hover over info icons for extra details).
  • Configuration – Configure your scenario parameters using visually appealing level cards.
  • WTP Results – View charts showing willingness-to-pay estimates for each attribute.
  • Predicted Programme Uptake – See the predicted uptake percentage with recommendations.
  • Costs & Benefits – Analyse cost components and outcomes (QALY-based and WTP-based) with interactive charts.
  • Saved Scenarios – Compare and export your saved configurations.

How to Use This Tool

  1. Review the Attributes tab to understand each programme feature. Hover over the info icons for details and reference levels.
  2. Configure your scenario in the Configuration tab using the level cards. All features (except method) are required; if no method is selected, it defaults to in-person.
  3. Click Apply Configuration to generate the results, then select View Results Summary for an on-screen summary of your scenario’s outcomes.
  4. Examine the WTP Results tab for the monetary value assigned to each attribute.
  5. Check the Predicted Programme Uptake tab for the uptake percentage and tailored recommendations.
  6. Review the Costs & Benefits tab for a detailed breakdown, interactive charts, and benefit-cost analysis (both QALY- and WTP-based).
  7. Save and compare scenarios in the Saved Scenarios tab, or export the results to PDF for sharing.

Willingness-to-Pay (WTP) Results

The willingness-to-pay (WTP) for each programme attribute is calculated by taking the ratio of that attribute’s coefficient to the negative of the cost coefficient. This quantifies the monetary value respondents place on each attribute, holding other factors constant.

For example, among programme types, community engagement is most valued with a WTP of A$14.47 (SE=3.31, p < 0.001). This means older adults are willing to pay an additional A$14.47 per session for community-based programs compared to a peer support program. Negative WTP values (e.g., -A$11.69 for fully virtual delivery) indicate how much cost would need to be reduced for that feature to be acceptable.
